---
title: "Add Integration"
description: "Configure integrations to add external SaaS platform data to PreNotify"
---

Integrations allow the addition of integration data for existing resources in your environment from external SaaS platforms like Microsoft Entra, Microsoft Intune, etc. Instead of manually managing separate resource records between platforms, users can correlate, review, and audit in one platform.

## Supported Integrations

These providers can be used to connect and sync to PreNotify:

- Microsoft Entra ID
- Microsoft Intune

### Microsoft Entra ID

### Microsoft Intune

## How to Configure Integrations

<Steps>
  <Step title="Navigate to Integration" stepNumber={1}>
    Click on the "Integrations" tab on the navbar.
  </Step>
  <Step title="Configure Integration" stepNumber={2}>
    Click the “Edit” button for the integration, under Actions in the table. On the new page, enter configuration settings and then click "Save" (ensure to enable the integration).
  </Step>
  <Step title="Sync Integration">
    When enabled and configured, on the integrations page, you will have the option to sync the integrations.
  </Step>
</Steps>
