## PreNotify

PreNotify is an enterprise-ready resource expiration notification dashboard. It centralizes resources from cloud and security integrations (such as Microsoft Entra ID and Microsoft Intune), tracks expiration dates and contact methods, and helps you avoid missed renewals with a unified dashboard and optional SSO.

## Key Features

- **Resource expiration tracking**: View total resources, expiring soon (30 days), expired, and resources without contacts
- **Unified dashboard**: Single view across integrations with filtering by integration type and resource type
- **Cloud integrations**: Sync resources from Microsoft Entra ID and Microsoft Intune
- **SSO (optional)**: Sign in with Microsoft Entra ID
- **Background sync**: RQ workers for scheduled and on-demand sync tasks
- **Enterprise-ready**: Built with Django, PostgreSQL, and Redis for scalable deployments
